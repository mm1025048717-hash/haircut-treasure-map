## 🎯 产品愿景

> **让每一次理发都不再是赌博，而是一次确定性的美好体验。**

在中国，有超过 **100万家** 理发店，但消费者找到一家靠谱的理发店却难如登天：

- 😫 大众点评刷单严重，评价失真
- 💸 办卡后发现Tony老师技术堪忧
- 🔄 换一个城市，又要重新踩雷
- 📱 小红书种草的店，实际体验差距巨大

**理发藏宝图** 要解决的就是这个痛点——通过真实用户的理发记录和口碑，帮助你快速找到附近技术好、性价比高的理发店和理发师。

---

## ✨ 核心功能

### 🗺️ 地图发现
基于地理位置，一键发现附近的理发店，支持按类型筛选（社区店、连锁店、高端店）。

### 📝 理发记录
记录每次理发的店铺、理发师、价格、效果，形成个人理发档案，方便复购和推荐。

### ⭐ 真实评价
只有真实消费过的用户才能评价，杜绝刷单水军，还原真实口碑。

### 💰 价格透明
显示店铺的真实价格区间，不再被办卡话术忽悠。

### 📌 收藏同步
收藏喜欢的店铺，即使换城市也能查看历史记录。

### 🔗 外部笔记聚合
整合小红书、大众点评等平台的真实用户笔记，多维度了解店铺。

## 🚀 快速开始

### 在线体验

直接访问：**https://haircut-treasure-map.vercel.app**

### 本地开发

```bash
# 克隆项目
git clone https://github.com/mm1025048717-hash/haircut-treasure-map.git
cd haircut-treasure-map

# 安装依赖
npm install

# 启动开发服务器
npx expo start

# 选择运行平台
# - 按 w 运行 Web 版本
# - 按 a 运行 Android 模拟器
# - 按 i 运行 iOS 模拟器
```

### 配置说明

1. **Supabase 配置**（可选，用于云端数据同步）
   ```bash
   cp .env.example .env
   # 编辑 .env 文件，填入你的 Supabase 配置
   ```

2. **高德地图配置**（可选，用于地图显示）
   - 在 [高德开放平台](https://lbs.amap.com/) 申请 Web 端 Key
   - 配置到环境变量 `EXPO_PUBLIC_AMAP_KEY`

# 华为应用市场上架申请填写指南

本文档将指导您完成华为应用市场的上架申请，解决所有必填项和错误提示。

---

## 📋 需要填写的5个部分

根据错误提示，您需要完成以下5个部分：
1. ✅ 隐私声明
2. ✅ 隐私标签信息录入
3. ✅ 版权信息
4. ✅ 备案信息
5. ✅ 联系方式

---

## 1️⃣ 隐私声明

### 1.1 隐私政策网址（必填）

**填写内容：**
```
https://haircut-treasure-map.vercel.app/privacy-policy
```

**说明：**
- 您需要先将 `PRIVACY_POLICY.md` 文件部署到可公开访问的网址
- 建议部署到 Vercel 或其他静态托管平台
- URL 必须以 `http://` 或 `https://` 开头

**操作步骤：**
1. ✅ 项目已包含 `public/privacy-policy.html` 文件
2. 部署到 Vercel 后，访问：`https://haircut-treasure-map.vercel.app/privacy-policy.html`
3. 在表单中填写该 URL

**部署方法：**
```bash
# 如果还没有部署，执行以下命令：
git add .
git commit -m "Add privacy policy page"
git push

# Vercel 会自动部署，部署完成后访问：
# https://haircut-treasure-map.vercel.app/privacy-policy.html
```

**如果使用其他域名：**
- 将 URL 中的 `haircut-treasure-map.vercel.app` 替换为您的实际域名
- 确保 URL 可以公开访问

### 1.2 隐私权利（可选）

**填写内容：**
```
https://haircut-treasure-map.vercel.app/privacy-rights
```

**说明：**
- 这是可选字段
- 如果暂时没有单独的隐私权利页面，可以留空或填写与隐私政策相同的 URL

---

## 2️⃣ 隐私标签信息录入

### 2.1 是否涉及个人信息收集

**选择：** ✅ **是**

**原因：** 应用需要收集位置信息来展示附近理发店，属于个人信息收集。

### 2.2 业务场景选择

根据应用功能，建议勾选以下场景：

- ✅ **应用功能**（已勾选）
- ✅ **产品个性化**（已勾选）
- ❌ 分析（如果只是基础统计，可不选）
- ❌ 广告与营销（当前版本不涉及）
- ❌ 向第三方披露（不涉及）
- ❌ 跨境传输（数据存储在国内）
- ❌ 其他用途（不涉及）

### 2.3 为业务场景添加数据项

您需要为"应用功能"和"产品个性化"两个场景添加数据项。

#### 场景1：应用功能

点击 **"+添加"** 按钮，添加以下数据项：

**数据项1：位置信息**
- **数据类别：** 位置信息
- **数据项：** 精确位置信息
- **数据说明：** 用于在地图上展示附近的理发店，仅在用户授权后获取

**数据项2：设备信息**
- **数据类别：** 设备信息
- **数据项：** 设备标识符
- **数据说明：** 用于优化应用体验和问题排查

#### 场景2：产品个性化

点击切换到"产品个性化"标签页，添加：

**数据项1：位置信息**
- **数据类别：** 位置信息
- **数据项：** 精确位置信息
- **数据说明：** 用于根据用户位置推荐附近的理发店

**数据项2：用户内容**
- **数据类别：** 用户内容
- **数据项：** 理发记录、收藏信息
- **数据说明：** 用于为用户提供个性化的理发店推荐和历史记录管理

## 3️⃣ 版权信息

### 3.1 电子版权证书（可选但推荐）

**操作步骤：**
1. 点击"浏览"按钮
2. 上传 PDF 格式的电子版权证书
3. 文件大小不超过 5MB

**如果没有电子版权证书：**
- 点击页面上的"【易版权】电子版权认证证书申请通道"链接
- 按照指引申请电子版权证书
- 如果已有纸质《软件著作权登记证书》，可以通过"补领通道"免费补领电子版

### 3.2 应用版权证书或代理证书（必填）

**操作步骤：**
1. 点击三个上传框中的任意一个（或多个）
2. 上传应用版权证书或代理证书
3. 支持格式：JPG、PNG、BMP
4. 单个图片不超过 15MB

**如果没有版权证书：**
- 需要先申请软件著作权登记证书
- 可以通过"【官方合作渠道】办理版权资质"链接了解详情
- 或者联系专业的知识产权代理机构

---

## 4️⃣ 备案信息

### 4.1 应用版权证书或代理证书（必填）

**操作步骤：**
1. 点击上传框（红色边框表示必填）
2. 上传证书图片
3. 格式：JPG、PNG、BMP
4. 单个图片不超过 15MB

**注意：** 如果第3步已经上传，这里可能也需要再次上传或确认。

### 4.2 授权书及其他材料（可选）

**操作步骤：**
1. 点击"上传"按钮
2. 上传 ZIP 格式压缩包
3. 文件大小不超过 200MB
4. 压缩包内可包含：
   - JPG、JPEG、BMP 格式图片
   - PDF 格式文件
   - 文件夹层级最多3层
   - 累计不超过200个文件

**说明：** 如果应用有授权书、营业执照等相关材料，可以打包上传。

### 4.3 APP类型信息（必填）

**根据您的应用情况选择：**

**选项1：您的APP服务器在中国大陆** ✅（推荐）
- 如果使用 Supabase 或其他国内服务器
- 如果数据存储在国内

**选项2：您的APP服务器不在中国大陆**
- 如果使用海外服务器（如 Supabase 海外节点）

**选项3：您的APP为单机APP**
- 如果应用完全离线运行，不需要服务器

**建议：** 根据您的 Supabase 配置选择。如果使用 Supabase 中国区或数据存储在国内，选择选项1。

## 5️⃣ 联系方式

### 5.1 应用负责人手机号码（必填）

**填写格式：**
- 国家代码：+86（已固定）
- 手机号码：11位数字
- 示例：`13800138000`

**说明：** 用于华为审核团队联系您，沟通应用审核问题。

### 5.2 应用负责人邮箱（必填）

**填写格式：**
- 有效的邮箱地址
- 示例：`your-email@example.com`
- 最大长度：500字符

**说明：** 用于接收审核结果通知和重要信息。

### 5.3 应用负责人姓名（必填）

**填写格式：**
- 真实姓名或负责人姓名
- 最大长度：256字符
- 示例：`张三` 或 `李四`

**说明：** 用于沟通时核实负责人身份。

## ✅ 填写完成后的操作

1. **保存草稿**
   - 点击右上角"保存"按钮
   - 保存当前填写的内容

2. **检查所有部分**
   - 确保5个部分都已填写完整
   - 确保没有红色错误提示

3. **提交审核**
   - 点击"提交审核"按钮
   - 等待华为审核团队审核

## 📝 常见问题

### Q1: 隐私政策网址还没有部署怎么办？

**A:** 您可以：
1. 快速部署到 Vercel：
   - 在项目中创建 `public/privacy-policy.html`
   - 将 `PRIVACY_POLICY.md` 内容转换为 HTML
   - 部署到 Vercel
   - 访问 `https://your-domain.vercel.app/privacy-policy.html`

2. 使用 GitHub Pages：
   - 创建 GitHub 仓库
   - 上传 HTML 文件
   - 启用 GitHub Pages
   - 获取公开 URL

### Q2: 没有版权证书怎么办？

**A:** 
- 需要先申请软件著作权登记证书
- 可以通过华为提供的官方合作渠道申请
- 或者联系知识产权代理机构
- 申请周期通常为1-3个月

### Q3: 备案信息中的"APP服务器位置"如何确定？

**A:**
- 查看您的 Supabase 项目设置
- 如果使用 Supabase 中国区，选择"在中国大陆"
- 如果不确定，可以查看 Supabase 控制台的项目信息
- 或者联系 Supabase 客服确认

### Q4: 隐私标签信息录入中的数据项如何填写？

**A:**
- 根据应用实际收集的数据填写
- 如果应用只收集位置信息，就只添加位置信息相关数据项
- 数据说明要清晰说明收集目的和使用方式
- 可以参考华为提供的示例和指南
